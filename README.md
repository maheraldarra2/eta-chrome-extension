# ETA Chrome Extension

A Chrome extension built to automate and simplify tasks on the ETA portal.  
This project demonstrates practical experience with Chrome APIs, content scripts, DOM manipulation, and extension architecture.

---

## 🚀 Features
- Automates repetitive actions on the ETA portal  
- Extracts and processes data directly from the page  
- Clean popup interface for quick actions  
- Uses content scripts to interact with DOM elements  
- Organized file structure and maintainable code  

---

## 📁 Project Structure
- `manifest.json` – Extension configuration  
- `content.js` – Logic injected into the webpage  
- `popup.html` / `popup.js` – User interface  
- `background.js` – Background logic (if used)  
- `icons/` – Extension icons  

---

## 🛠️ Technologies Used
- JavaScript  
- Chrome Extensions API  
- HTML / CSS  
- DOM Manipulation  

---

## 📦 How to Install (Developer Mode)
1. Download the project as ZIP  
2. Extract the folder  
3. Open Chrome → Extensions  
4. Enable **Developer Mode**  
5. Click **Load unpacked**  
6. Select the extracted folder
